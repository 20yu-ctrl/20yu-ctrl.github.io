GitHub 的使用

首先访问 GitHub 官网，点击 Sign up 注册一个账号，注册成功后登录进去，点击右上角头像旁边的 + 号并选择 New repository 菜单进行创建仓库

![](./images/git/图片 53_2.png)

填写仓库基础信息

![](./images/git/图片 58.png)

获取远程仓库的 url 地址

![](./images/git/图片 54_2.png)

克隆远程仓库到本地

在工作空间项目目录下空白位置右键，并选择 Git Clone… 菜单

![](./images/git/图片 55_2.png)

填写刚刚创建的远程仓库 url 地址，以及选择仓库下载到哪个目录

![](./images/git/图片 56_2.png)

点击ok即可将远程仓库克隆到本地了，当然你也可以通过命令

这个过程需要分成三部，分别为 add/commit/push
